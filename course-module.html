Scratch Programming Course Curriculum

Comprehensive guide to teaching computational thinking through creative projects

*Module 6 & 7 PPT Only Completed*

Course Modules

Module 1: Introduction to Scratch

Topics Covered: Scratch Installation, Interface Familiarization, Motion Blocks, Looks and Sound Blocks

Project Learning Goal
Hello World (PPT) Familiarize students with Scratch interface, sprites, backdrops, and blocks.
Sprite Walks Across the Screen (PPT) Learn to make sprites move in different directions.
Dancing Sprite (PPT) Add sounds and change sprite costumes.

Module 2: Events and Sequencing

Topics Covered: Using Events, Sequencing Actions, Introducing Loops

Project Learning Goal
Sprite Reacts to Mouse Clicks (PPT) Learn how to trigger actions using events.
Dance Routine (PPT) Understand how to sequence multiple actions.
Animate a Bouncing Ball (PPT) Understand loops and repetition in coding.

Module 3: Animations and Storytelling

Topics Covered: Simple Animations, Broadcast Messages

Project Learning Goal
Create a School Story Scene (PPT) Animate multiple sprites to tell a story.
Broadcast Messages (PPT) Learn how sprites communicate via broadcasts.

Module 4: Variables and Conditions

Topics Covered: Introduction to Variables, Operators, Introduction to Conditions

Project Learning Goal
Frog Game (PPT) Learn about variables and tracking values like scores.
Create a Math Quiz Game (PPT) Understand arithmetic and comparison operators.
Even or Odd Checker (PPT) Use "if" and "if-else" statements for logic.

Module 5: Decision Making and Game Design

Topics Covered: Advanced Conditions, Game Mechanics, Multi-Level Games

Project Learning Goal
Student Grade System (PPT) Combine multiple conditions for complex logic.
Banking Project (PPT) Implement a simple banking system with deposits/withdrawals.
Catching Game (PPT) Use variables for time and levels in a game.
Maze Master (PPT) Navigate a series of mazes; each level adds moving walls or traps.

Module 6: Variables, Lists & Data

Topics Covered: Working with Lists, Variables and Logics, Unit Converter

Project Learning Goal
Build a Quiz Game (PPT) Store and use multiple values with lists.
Create a Calculator App (PPT) Use variables, Operators, Logics.
Temperature Converter (PPT) Convert units like °C→°F with input blocks and math.

Module 7: Advanced Concepts

Topics Covered: Custom Blocks, Keyboard Access, Add & Remove Blocks, Extension

Project Learning Goal
Electrical Circuit (PPT) Create reusable blocks for efficiency (Touching Color).
Typing Practice Game (PPT) Improve typing speed with sprite feedback.
To-Do List App (PPT) Add/remove tasks using list blocks.
Language Translator (PPT) Use lists to translate words from one language to another.

Module 8: Final Project

Topics Covered: Desktop UI, 2D Platformer Engine, Artificial Intelligence

Project Learning Goal
Scratch OS (PPT) Multiple screens, draggable windows, apps.
Jump Quest (PPT) Gravity simulation, jump arc, collision detection, tile-based level system.
Chess Game with AI Opponent (PPT) Chess rules implementation: movement, capture, turn-switching.
